# 又拍使用说明

#### 一、在appsettings.json中添加配置项
```
"UpYunSettings": {
    "BucketName": "yihemed-weixin",
    "UserName": "yihemed",
    "Password": "yihemed123",
    "Domain": "http://yihemed-weixin.test.upcdn.net"
  },
```

#### 二、在Startup中注册又拍配置
```
services.Configure<UpYunSettings>(
	Configuration.GetSection("UpYunSettings"));
```
